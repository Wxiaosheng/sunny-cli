# `@sunny-cli/core`

> This is my cli.

## 架构
### core 模块
1. 准备阶段
2. 注册命令
3. 执行命令

#### 准备阶段
1. 检查 cli 版本
2. 检查 node 版本
3. 检查用户主目录
4. 检查是否 root 启动
5. 检查入参
6. 全局检测是否需要提示更新

#### 注册命令
1. 通过 commander 库注册命令
2. 封装 Command 类，将不同的命令抽离成单独的命令库
3. 封装 @sunny-cli/exec 多进程 动态执行 远程命令库

#### 执行命令
1. 封装 Package 类，用于管理远程 npm 包
2. 安装并缓存远程命令包
3. ejs 模板渲染
4. 自动执行模板对应的安装、启动命令
